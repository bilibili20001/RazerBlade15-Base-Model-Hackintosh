## 已知问题：

1.隔空投送只能单向传输，手机到电脑没问题，电脑到手机有问题，看远景貌似是这款网卡的通病，暂时无解

2.开机默认耳机输出，扬声器外放需要手动切换下，如需自动切换使用蓝牙耳机即可

3.外接显示器无声音输出
